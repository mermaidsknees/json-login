# json-login



Features: 

  - Firebase passwordless login system
  - Private routes
  - CRUD operations using Axios
  - Infinite Scrolling
 



For the site to properly run locally, it must run on ports 8080 and 3000


## Project setup
```
npm install
npm run serve
```
## For live server
```
cd public/data
json-server users.json
```
